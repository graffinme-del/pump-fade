# PUMP FADE — AI Trading Terminal

> Торговый терминал для поиска сетапов на крипторынке с AI-верификацией через Gemini

**URL:** https://pump-fade.vercel.app  
**GitHub:** graffinme-del/pump-fade  
**Stack:** Vanilla JS · Binance Futures API · Gemini AI · Lightweight Charts · Vercel

---

## Что это такое

Pump Fade — браузерный торговый терминал который автоматически сканирует топ-200 монет на Binance Futures, находит торговые сетапы по нескольким алгоритмам, верифицирует их через Gemini AI и отслеживает результаты.

---

## Архитектура

```
Binance Futures API (fapi.binance.com)
    ↓
СКАНЕР (каждые 3 мин) — 200 монет, батчами по 12
    ↓
scoreForIdea() — 5 TF анализ + паттерны свечей
    ↓
Gemini AI Verdict (GO / WAIT / SKIP)
    ↓
GO → Сигналы | WAIT → Наблюдение | SKIP → удалить
    ↓
Winrate tracker (проверка каждые 15 мин)
```

## Детекторы сетапов

### LONG сетапы (5 типов)

| Тип | Условие | Источник |
|-----|---------|----------|
| 🚀 Старт пампа 5m | Тело 0.35-3%, Vol×2+/мед, quality ≥72 | `early_pump_scanner.py` |
| 📈 EMA стек ↑ | close>EMA20>EMA50 на 1h + Vol + CVD ↑ | `h1_context.py` |
| 📈 Пробой EMA20 ↑ | Свечи ниже → выше EMA20 на 15m | Оригинальный |
| ↩ Разворот снизу | ATR≥0.22%, ROC≤-1.8% + бычья свеча | `reversal_tf_pair.py` |
| 📈 Volume Breakout | Vol×2.5+, бычья свеча, RSI 50-75 | Оригинальный |
| 📉 RSI перепроданность | RSI 1h<32, 15m<38 + MACD ↑ | Оригинальный |

### SHORT сетапы (6 типов)

| Тип | Условие | Источник |
|-----|---------|----------|
| ▼ Pump Exhaustion | Памп>15%, dist>8% + RSI/MACD/CVD сигналы | Основная стратегия |
| 📉 Разворот тренда ↓ | close<EMA20<EMA50 только что выровнялся | `h1_context.py` |
| 📉 Пробой EMA20 ↓ | Свечи выше → ниже EMA20 с объёмом | Оригинальный |
| RSI перекупленность | RSI>72 + MACD ↓ + CVD падает | Оригинальный |
| ↩ Разворот сверху | ATR≥0.35%, ROC≥2.2% + отказная свеча | `reversal_tf_pair.py` |
| 📉 Пробой поддержки | Vol×2+, медвежья свеча ниже EMA20+EMA50 | Оригинальный |

## МТФ анализ (H1_ORIENT_SCORE_ADJUST)

Каждый сигнал корректируется по согласованности таймфреймов:

```
4h согласован с направлением:  +10 к score
4h конфликтует:                -15 к score
1h согласован:                  +8 к score  
1h конфликтует:                -10 к score
```

Таймфреймы в карточке монеты: `5m · 15m · 1h · 4h`

## Паттерны свечей

Добавляют бонус/штраф к скорингу (±8-14 пунктов):

**Медвежьи:** Доджи, Повешенный, Медвежье поглощение, Падающая звезда, Три вороны, Вечерняя звезда, Харами медвежий

**Бычьи:** Молот, Бычье поглощение, Утренняя звезда, Три солдата, Харами бычий, Перевёрнутый молот

## Gemini AI интеграция

**Модель:** `gemini-2.5-flash`  
**Ключ:** бесплатно на aistudio.google.com

### Вердикт при сканировании
- Промпт на английском (экономия токенов кириллицы)
- Ответ: `{"verdict":"GO","confidence":80,"reason":"..."}`
- **GO** → сигнал остаётся, звук + push уведомление
- **WAIT** → монета уходит в "Наблюдение", перепроверка каждые 5 мин
- **SKIP** → тихо удаляется

### Кнопка AI анализ
- Если вердикт уже есть → объясняет подробнее почему GO/WAIT
- Если нет → делает полный анализ
- Ответ в правой панели чата

---

## Winrate трекер

- Записывает каждый сигнал с Entry/TP/SL в localStorage
- Минимальные уровни: **TP ≥5%** от entry, **SL ≥2.5%**, **R/R = 2.0**
- Проверка цен каждые **15 минут**
- При достижении TP или SL → фиксирует результат + push уведомление
- Записи старше 48ч → помечаются как `expired`

### Карточка монеты содержит:
- Вердикт Gemini (GO/WAIT) с причиной
- Entry / Stop-loss / TP зона / Профит R:R
- Памп 8h, Скорость, Vol/avg, RSI, ATR ratio, CVD
- CVD 15m (тренд дельты, дивергенция)
- Фьючерсы (Open Interest, Funding Rate)
- МТФ бейджи (5m/15m/1h/4h)
- Фазы истощения (0/3 → 3/3)
- Паттерны свечей
- Кнопки: TradingView ↗ · ✦ AI анализ

|------|---------|

## Технические детали

**API:** `fapi.binance.com/fapi/v1` (Futures, без CORS)  
**WS:** `stream.binance.com:9443` (топ-20 монет реальное время)  
**Данные на монету:** 1h×120 + 15m×80 + 4h×60 + 5m×60 свечей  
**Дедупликация:** 2 часа (не показывать ту же монету повторно)  
**Хранение:** localStorage (`pf_wr_v2` для winrate)  

**Скоринг качества (0-100):**
- ≥72 → сигнал выдаётся
- ≥80 + нет МТФ конфликта → сразу GO без ожидания Gemini
- <72 → игнорируется
